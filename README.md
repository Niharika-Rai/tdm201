# TDM 201
A portfolio of Data Mine Seminar Projects for Purdue's TDM 201 class. 

I took TDM 201 in the Fall 2025 semester at Purdue, which was a mandatory requirement for the Data Mine Corporate Partners Program. Since this was my second year at the Data Mine (the 1st was TDM 102), I took the second-year, 1-credit course.
